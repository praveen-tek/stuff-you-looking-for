# Student Resource Hub

A website for MSME students to find teacher-recommended books, notes, and required materials for each subject all in one place instead of scattered across group chats.

## Idea

- Every subject/course has a page listing:
  - Recommended books (with author, edition, and where to buy/download)
  - Required materials (drafting kits, lab equipment, stationery, etc.)
  - Notes/PYQs shared by seniors or teachers
- Students and friends can browse by semester, subject, or course code.
- Content is contributed by students themselves — crowdsourced and kept updated each session.

## Contributing

1. Fork the repo and clone your fork.
2. Create a new branch:
   ```bash
   git checkout -b add-subject-<name>
   ```
3. Add or update content:
4. Follow the existing entry format (title, author/brand, link, notes).
5. Commit with a clear message:
   ```bash
   git commit -m "Add Physics-I recommended books"
   ```
6. Push and open a Pull Request against `main`.
7. One reviewer approval required before merge.

### Contribution Rules

- Only add books/materials actually recommended by a teacher or verified by multiple students.
- No pirated PDF links — link to official sources or list where to buy.
- Keep entries short and factual (no personal opinions in the data files).
- Open an issue first for big structural changes (new sections, redesign, etc.).
